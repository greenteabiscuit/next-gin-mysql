# next-gin-mysql

## For Development
Place the `.env` file, one under `backend`, another under `migration`

```
HTTP_HOST=""
HTTP_PORT=8080
DB_HOST="mysql"
DB_PORT=3306
DB_USERNAME=root
DB_PASSWORD=root
DB_DATABASE=backend
```

`.air.toml` can be found in the [air repository](https://github.com/cosmtrek/air)
